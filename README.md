### Installation

<div>
    <div>
        If you are using Yarn as your package manager, consider using the following command:
        <code>
            yarn add @lemooos/ui
        </code>
    </div>
    <div>
        But if you prefer to use npm <em>(root feelings)</em>, consider using <em>this</em> command:
        <code>
            npm i @lemooos/ui
        </code>
    </div>
</div>
