### Installation

<div>
<div>
    If you are using Yarn as your package manager, consider using the following command:

<code>
    yarn add @lemooos/ui
</code>
</div>

<div>
But if you prefer to use npm *(root feelings)*, consider using *this* command:
<code>
    npm i @lemooos/ui
</code>
</div>
